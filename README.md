# ⚡ Electrical Grid Stability Prediction

An AI-powered web application that predicts the stability of an electrical grid using machine learning.

## 🚀 Overview

This project combines a React frontend and FastAPI backend to provide real-time electrical grid stability predictions. Users can input grid parameters and instantly receive stability predictions generated by a trained CatBoost model.

## ✨ Features

* Interactive React dashboard
* FastAPI-powered prediction API
* CatBoost machine learning model
* Real-time stability prediction
* Responsive user interface
